# GitHub 推送命令

当您获得有效的 token 后，运行以下命令：

## 1. 设置远程仓库（使用新 token）
```bash
git remote set-url origin https://YOUR_NEW_TOKEN@github.com/tiaochao/YouTubeRadar.git
```

## 2. 推送代码
```bash
git push -u origin main
```

## 3. 如果需要强制推送
```bash
git push -u origin main --force
```

## 验证 Token 是否有效
```bash
curl -H "Authorization: token YOUR_TOKEN" https://api.github.com/user
```

## 当前状态
- Git 仓库已初始化 ✓
- 第一次提交已创建 ✓
- 等待有效的 token 推送到 GitHub

## Token 权限要求
创建 token 时需要勾选：
- [x] repo (Full control of private repositories)
  - [x] repo:status
  - [x] repo_deployment
  - [x] public_repo
  - [x] repo:invite
  - [x] security_events
