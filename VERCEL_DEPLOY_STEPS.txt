Vercel 部署步骤
===============

1. 在终端运行登录命令：
   vercel login

2. 选择登录方式（推荐 GitHub）

3. 登录成功后，部署到生产环境：
   vercel --prod

4. 第一次部署会询问：
   - Set up and deploy "~/YouTubeRadar"? 选 Y
   - Which scope? 选择您的账号
   - Link to existing project? 选 N（如果是新项目）
   - What's your project's name? 输入 youtube-radar 或保持默认
   - In which directory is your code located? 按回车（使用当前目录）
   - Want to override the settings? 选 N

5. 等待部署完成，会获得：
   - Production URL: https://your-app.vercel.app

备选方案：通过网页部署
====================
如果 CLI 有问题，直接访问：
https://vercel.com/import/git?s=https://github.com/tiaochao/YouTubeRadar

这样更简单，点击几下就能完成部署。
